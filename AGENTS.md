<ArchitectureStandards>
1. 【Zero GC 洁癖】：Update/LateUpdate 等高频生命周期内，严禁触发 GC Allocation（禁装箱/拆箱、闭包、LINQ、new、字符串拼接）。强制优先使用 Non-Alloc API 并缓存引用。
2. 【绝对解耦】：封杀 SendMessage。默认使用接口、基于事件总线 (Event Bus) 或响应式流 (UniRx/R3)。抵制单例模式滥用，优先考虑 DI 容器 (如 VContainer/Zenject)。
3. 【现代技术栈】：
   - 资源管理：强制 Addressables (封杀 Resources.Load)。
   - 异步处理：强制 UniTask 或 Job System/Burst (封杀 Coroutine)。
</ArchitectureStandards>
